# K-2SO Animatronic Sketch v1.1.0 -- Archive (read-only)

This is the **original first sketch** of the K-2SO Animatronic Controller, kept here as
a historical reference. **Do not flash this version on a current build.** Use v1.2.4 or
v1.3.0 instead.

## What this version has

- Basic 4-servo control (Eye Pan/Tilt, Head Pan/Tilt)
- Simple eye animation (single mode)
- DFPlayer Mini audio playback
- Rudimentary serial CLI

## What it does NOT have

- WiFi web interface
- IR learning remote
- Status LED feedback
- Personality modes
- Sequence recording / playlist / IR mapping
- EEPROM configuration with profiles
- Most stability fixes from later versions

## Documentation

Use the System Documentation PDF in the project root for current usage:
`generate docs/K-2SO_Animatronic_Controller_System_Documentation_v1.3.0.pdf`

The original v1.1.0 README is preserved as `README.md.bak` next to this file.
